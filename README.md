## Getting Started

This project uses [Flutter](https://flutter.dev/) and the web build is continously deployed on [Vercel](https://vercel.com/). The current CI solution is [Codemagic](https://codemagic.io/).

## Setup

```bash
flutter pub get
flutter run
```
